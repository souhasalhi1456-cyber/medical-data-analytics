# Mise en place d’un système décisionnel pour l’analyse de données médicales

## Présentation du projet
Ce projet a été réalisé dans le cadre de mon parcours en **Business Computing**.  
Il porte sur la conception d’un **système décisionnel** dédié à l’analyse de données médicales afin de mieux comprendre les facteurs qui influencent la santé des individus.

L’objectif est de transformer des données brutes issues d’un jeu de données Kaggle en informations utiles, à travers un processus complet allant du **nettoyage des données** jusqu’à la **visualisation des résultats**.

## Objectifs du projet
- Nettoyer et préparer un jeu de données médicales.  
- Identifier les variables les plus pertinentes liées au mode de vie et à la santé.  
- Concevoir un modèle de données (schéma en étoile) pour le stockage et l’analyse.  
- Réaliser un tableau de bord décisionnel pour interpréter les résultats.  

## Outils utilisés
- **Python / Jupyter Notebook** : analyse exploratoire, traitement et préparation des données.  
- **Pandas, NumPy, Matplotlib, Scikit-learn** : manipulation et visualisation des données.  
- **Talend & PostgreSQL** : création du data warehouse.  
- **Power BI** : conception du tableau de bord final et interprétation des indicateurs.  


## Contenu du dépôt
- `projet pi sem2 1ere annee final.ipynb` : le notebook Jupyter contenant le travail de préparation et d’analyse.  
- `health_data.csv` *(ou ton fichier CSV original)* : le jeu de données utilisé dans le projet.  
- *(optionnel)* : captures d’écran du tableau de bord Power BI ou du modèle de données.  



## Résultats et enseignements
L’analyse a permis d’identifier plusieurs corrélations entre les habitudes de vie (activité physique, sommeil, alimentation) et certains indicateurs de santé.  
Le système décisionnel mis en place facilite la visualisation de ces tendances et peut servir de base pour des analyses médicales plus poussées.
